### Prompts utilizados para construir la documentación del sistema LTI

#### Prompt del usuario

```
Necesito realizar una actividad en la que debo diseñar y documentar un sistema de software llamado LTI. Es un sistema ATS (Applicant Tracking System) que debe incluir fases de análisis, modelado, diseño de alto nivel, casos de uso, y más. Me piden documentarlo todo en un archivo markdown y subirlo a un repositorio colaborativo.
¿Puedes ayudarme paso a paso?
```

#### Respuesta de ChatGPT

Se entregó una guía estructurada que incluye:

* Redacción del resumen del sistema y valor diferencial
* Lean Canvas en markdown
* Descripción de los casos de uso principales
* Modelo de datos relacional básico
* Arquitectura del sistema explicada
* Componente del sistema detallado como Diagrama C4
* Recomendaciones para generación de diagramas y estructura de entrega

Todos estos elementos fueron consolidados en un archivo `LTI-DR.md`.
